Loan Acceptance Prediction System

Problem Statement

Financial institutions face high risk in approving loans. This project predicts loan approval eligibility based on applicant details to reduce financial loss.

Tech Stack

  • Python
  • Pandas, NumPy
  • Scikit-learn

Techniques Used

  • Exploratory Data Analysis (EDA)
  • Feature Engineering
  • Classification Algorithms

Outcomes

  • Achieved 85% accuracy on validation data
  • Identified credit history as a key approval factor
  • Designed for future Streamlit deployment
